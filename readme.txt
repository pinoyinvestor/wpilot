AI assistant for WordPress. Build, optimize, and manage your site through conversation.

== Description ==

WPilot is an AI assistant inside your WordPress dashboard. Chat naturally and WPilot executes changes on your site.

== Installation ==

1. Upload the wpilot folder to /wp-content/plugins/
2. Activate the plugin
3. Go to WPilot > Settings and connect your API key
4. Start chatting
