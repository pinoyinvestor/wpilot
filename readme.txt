== Description ==

WPilot is a live AI assistant inside your WordPress dashboard. Connect your Claude API key and chat naturally — WPilot analyzes your site, executes changes, and handles everything from SEO to WooCommerce to security.

= What makes WPilot different =

* **Acts, doesnt ask** — Say fix my SEO and it fixes ALL pages. No unnecessary questions.
* **168 executable tools** — Pages, posts, SEO, CSS, images, WebP, WooCommerce, plugins, security, SMTP, caching, and more.
* **Automatic backup** — Every change has a one-click undo button. Crash recovery built-in.
* **Works with YOUR plugins** — Configures WooCommerce, Elementor, Rank Math, Wordfence, LiteSpeed Cache, Amelia, and 20+ more.
* **Page builder aware** — Creates real Elementor, Divi, or Gutenberg pages.
* **Gets smarter** — Brain memory system learns your site preferences over time.
* **25 languages** — Interface translated into 25 languages.

== Installation ==

1. Upload the wpilot folder to /wp-content/plugins/
2. Activate the plugin through the Plugins menu
3. Get a Claude API key from console.anthropic.com
4. Go to WPilot > Settings and paste your API key
5. Start chatting!

== Frequently Asked Questions ==

= Do I need a Claude API key? =
Yes. WPilot uses the Claude AI API. You need your own API key from Anthropic. This costs approximately /bin/bash.003-0.01 per message.

= Is my data safe? =
Your API key and all site data stays on YOUR server. WPilot never sends your API key, site URL, or personal data to our servers.

= Can WPilot break my site? =
Every change has an automatic backup. Click Undo to restore instantly. Crash recovery detects PHP errors and offers to restore.
