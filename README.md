# WPilot Lite — Powered by Claude

**Version: 1.0.0**

AI-powered WordPress management. Chat with your site, manage SEO, security, and design — powered by Claude.

## Features

- MCP integration for Claude Desktop
- OAuth 2.1 with PKCE authentication
- Dark theme admin UI with i18n (en/sv/de/fr/es)
- Server-side sandbox for safe code execution
- temp-file + include execution (wp.org compliant)
- Help tab with Lite vs Pro comparison

## Requirements

- WordPress 6.0+
- PHP 8.0+
- Claude Desktop with MCP support
